# 🔑 Random Password Generator

A sleek, lightweight web application designed to generate secure, random passwords. This tool focuses on simplicity and ease of use, featuring a one-click "copy to clipboard" functionality.

## 🚀 Features
- **Instant Generation:** Create random passwords with a single click.
- **Copy to Clipboard:** Modern clipboard integration using the `navigator.clipboard` API for a seamless user experience.
- **Clean UI:** A minimalist and intuitive interface built with accessibility in mind.
- **Pure JavaScript:** Lightweight and fast, using no external libraries (Vanilla JS).

## 🛠️ Built With
- **HTML5:** Semantic structure.
- **CSS3:** Modern styling and responsive layout.
- **JavaScript (ES6):** Logic for password generation and clipboard management.
